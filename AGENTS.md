# Repository Rules

- Treat this repository as a growing educational LLM knowledge base.
- Keep the root README concise and navigational.
- Put detailed implementation notes beside the relevant tool or workflow.
- Reuse canonical scripts instead of duplicating pipelines.
- Never commit tokens, private paths, private SSH aliases, raw logs, model weights, or large generated artifacts.
- Run privacy scans and repository-relative link checks before proposing a sync.
- Do not perform remote GitHub writes until the user explicitly sends the exact message `同步`.
- Do not force-push.
- Do not upload Hugging Face artifacts unless separately authorized.
